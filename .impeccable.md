## Design Context

### Users
Shanghai Gaokao candidates and their parents use this product while comparing admission options, checking rank fit, and reviewing historical cutoff trends. They need fast answers, confidence in the data, and clear paths from a ranking input to actionable groups of options.

### Brand Personality
Calm, precise, trustworthy. The interface should feel like a careful admissions workbench: measured, legible, and factual, with no promotional hero treatment or decorative distraction.

### Aesthetic Direction
Light mode only. Use a restrained editorial-data direction with tinted paper surfaces, compact controls, strong table readability, and muted but distinct safety colors. Start directly with functionality instead of a landing page.

### Design Principles
- Put the primary task first: rank input, year selection, and results should be immediately available.
- Treat data density as a feature: organize it with hierarchy, filters, and consistent spacing rather than hiding it behind decoration.
- Use calm color sparingly: accents should clarify state, not decorate.
- Make uncertainty explicit: disclaimers, 580+ limitations, and safety categories should read as guidance, not guarantees.
- Keep every page visually connected through the same surfaces, controls, and table language.
